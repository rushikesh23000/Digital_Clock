🕒 Digital Clock using Tkinter
📌 Project Overview

This is a simple Digital Clock application built using Python and the Tkinter GUI library.
It displays the current time and date, updating every second in real-time.

🚀 Features
⏰ Live digital clock (HH:MM format)
📅 Displays current date
🔄 Auto-refresh every second
🎨 Simple and clean user interface
🛠️ Technologies Used
Python
Tkinter (for GUI)
time module (strftime function)

💡 Code Explanation
Tk() → Creates main window
Label → Displays time and date
strftime() → Gets current time/date
after(1000, function) → Updates every 1 second
